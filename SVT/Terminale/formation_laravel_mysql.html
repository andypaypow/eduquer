Formation Laravel & MySQL

Apprendre à créer et connecter une base de données avec Laravel (sur WAMP)

Formation par Eric : +229 41757682

Objectif de la Formation

Ce que vous allez apprendre
  • Créer une base de données MySQL avec WAMP
  • Configurer Laravel pour s'y connecter
  • Vérifier la connexion en exécutant une migration
🎯

Prérequis

Assurez-vous d'avoir WAMP installé sur votre machine avant de commencer cette formation.

Guide Complet - Étape par Étape

1

Ouvrir WAMP et vérifier le serveur MySQL

Points de vérification
  • Lance WAMP
  • L'icône dans la barre doit être verte (tout fonctionne)
  • Clique dessus → choisis phpMyAdmin
  • Ou ouvre manuellement : http://localhost/phpmyadmin
2

Créer la base de données

Dans phpMyAdmin :

  1. Clique sur "Nouvelle base de données"
  2. Entre un nom, par exemple : laravel_site
  3. Laisse le jeu de caractères sur utf8mb4_general_ci
  4. Clique sur Créer
3

Configurer Laravel dans le fichier .env

Dans ton projet Laravel, ouvre le fichier .env à la racine.

Modifie la partie base de données comme ceci :

Fichier .env 
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel_site
DB_USERNAME=root
DB_PASSWORD=
4

Vérifier la configuration

Tu peux ouvrir config/database.php et vérifier que la section mysql est bien comme ceci :

config/database.php 
'mysql' => [
    'driver' => 'mysql',
    'host' => env('DB_HOST', '127.0.0.1'),
    'port' => env('DB_PORT', '3306'),
    'database' => env('DB_DATABASE', 'forge'),
    'username' => env('DB_USERNAME', 'forge'),
    'password' => env('DB_PASSWORD', ''),
    'charset' => 'utf8mb4',
    'collation' => 'utf8mb4_unicode_ci',
    'prefix' => '',
    'strict' => true,
    'engine' => null,
],
5

Tester la connexion avec une migration

Ouvre ton terminal dans le projet Laravel et exécute :

Commande :
php artisan migrate
Résultat attendu

🟢 Si tout est bien configuré : Laravel va créer automatiquement les tables :

  • migrations
  • users
  • password_reset_tokens
  • failed_jobs
En cas d'erreur

🔴 Si tu as une erreur du type "Access denied" ou "Unknown database", vérifie :

  • que WAMP est bien lancé
  • que le nom de ta base correspond exactement à celui du .env
  • que ton utilisateur et ton mot de passe MySQL sont bons

Support

Si tu as d'autres soucis, envoie juste une photo et je verrai mieux.

BOTANICA

📋 Étape 1 - Installations Préalables

WAMP installé et icône verte

🔧 Étape 2 - Créer le Projet Laravel

Terminal (PowerShell ou cmd) :
cd C:\wamp64\www

🗄️ Étape 3 - Créer la Base de Données

  1. Ouvrir http://localhost/phpmyadmin
  2. Nouvelle base → nom : laravel_site → interclassement : utf8mb4_general_ci → Créer

⚙️ Étape 4 - Configurer la Connexion

Fichier C:\wamp64\www\laravel_site\.env

.env 
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel_site
DB_USERNAME=root
DB_PASSWORD=

🏗️ Étape 5 - Créer la Table contacts

Terminal :
php artisan make:model Contact -m

Fichier de migration (créé dans database/migrations/) :

Migration contacts 
public function up()
{
    Schema::create('contacts', function (Blueprint $table) {
        $table->id();
        $table->string('name');
        $table->string('email');
        $table->text('message');
        $table->timestamps();
    });
}
Terminal :
php artisan migrate

🎮 Étape 6 - Créer le Contrôleur

Terminal :
php artisan make:controller ContactController

app/Http/Controllers/ContactController.php :

ContactController 
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use App\Models\Contact;

class ContactController extends Controller
{
    public function index()
    {
        return view('contact');
    }

    public function store(Request $request)
    {
        $request->validate([
            'name'    => 'required|max:255',
            'email'   => 'required|email',
            'message' => 'required'
        ]);

        Contact::create([
            'name'    => $request->name,
            'email'   => $request->email,
            'message' => $request->message
        ]);

        return back()->with('success', 'Merci pour votre message !');
    }
}

🛡️ Étape 7 - Protéger le Modèle

app/Models/Contact.php :

Contact Model 
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;

class Contact extends Model
